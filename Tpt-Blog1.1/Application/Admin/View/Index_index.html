欢迎使用Tpt-Blog博客管理系统，先温馨提醒几个常见问题：
程序正式上线运营请把index.php里面调试模式关闭； 确认服务器或空间开启了伪静态；
安装完成后请将根目录install文件和index.php里面安装代码删除； 请将程序内的所有文件直接放在根目录下，不要多层目录；
尽管本程序在发布前已经经过了若干严格测试，但我们依然强烈建议您随时备份； 程序安装好后一定记得修改密码和口令；
